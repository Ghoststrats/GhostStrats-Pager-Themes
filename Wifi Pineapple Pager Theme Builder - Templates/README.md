# GhostStrats

## Wi Fi Pineapple Pager Theme Builder

### Template Guide

## Included Folders

### GhostStrats

Theme with visible icon buttons.
Buttons must be customized manually.

### GhostStratsV2

Theme with buttons hidden.
This is the easiest option and recommended for beginners.

---

## Step 1 Download

For this demonstration, download the ZIP file directly from GitHub.

1. Click the GitHub ZIP download option
2. Extract the ZIP file
3. You should now see the template folders

Optional
You can also download the templates from the shared Proton Drive link if you prefer.

---

## Step 2 Dashboard Background V2 Only

1. Open the **GhostStratsV2** folder
2. Open the **asset** folder
3. Open the **dashboard** folder

Replace the file named
**gs_bg.png**

Your custom image must be sized to
**480 x 222**

### Recommended Method

1. Copy your custom image into the dashboard folder
2. Right click **gs_bg.png**
3. Select Rename and copy the filename
4. Delete **gs_bg.png**
5. Rename your custom image to **gs_bg.png**

---

## Step 3 Boot Animation

1. Go back to the **asset** folder
2. Open the **boot_animation** folder

Replace all four images using the same method as above.

This process must be repeated four times.
All filenames must match exactly.

---

## Optional Recon Screen Background

1. Download the Recon template from the shared Proton Drive
2. Customize the image
3. Upload it as
   **recon_dashboard.png**

Use the same replacement process described above.

---

## Step 4 Copy Theme to Pager Windows

Open **WinSCP**

**Host**
172.16.56.1

**Port**
22

**Username**
root

**Password**
Use the password you set during device setup

---

## Step 5 Install Theme

Navigate to
**root/mmc**

If there is no **themes** folder

1. Right click
2. New
3. Directory
4. Name it **themes**

Drag your theme folder into
**root/mmc/themes**

Wait for the transfer to finish.

---

## Step 6 Enable Theme

On the pager

1. Open Settings
2. Select Themes
3. Choose your theme

---
